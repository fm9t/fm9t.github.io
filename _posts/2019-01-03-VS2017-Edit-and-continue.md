# VS2017 调试时编辑并继续功能
VS2017安装好后在调试时如果修改代码会报Changes are not allowed while code is running错误，只需要将Tools->Options->Debug->General下的Enable Edit and Continue选项去掉即可在调试时编辑并继续。
